#### 進度: 未完成微調

- 文本預處理:
    - 加上向量模型微調
    - 文本、向量資料庫構建
    - 向量模型: sentence transformer
    - 目標: 關鍵字、關鍵句，餘弦距離越短越好，意味著在進行向量資料庫中搜尋，類似的句子會越相近
    - 方法: (Anchor, Positive, Negative)
        - Anchor (錨點): 文章的關鍵句向量
        - Positive (正樣本): 文章本體向量（或反過來，錨點是本體向量，正樣本是關鍵句向量）
        - Negative (負樣本): 來自不同文章的文章本體向量
        - 產生負樣本:
            - 隨機負採樣: 對於每一個 (anchor, positive) 對，從其他不同的文章中隨機選擇一個文章本體向量（或關鍵句向量）作為負樣本
            - 批內負採樣: 在一個訓練批次 (batch) 中，將批次內其他樣本的 positive 向量視為當前 anchor 的 negative 向量
